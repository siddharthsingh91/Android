1.Run the PhoneGap create command: Run “phonegap create ProjectPath PackageName ProjectName” 2.Switch your command line directory to your project directory: Run “cd ProjectPath” 3.Build for the Android platform: Run “phonegap build android” (This is important as it adds the android target to the PhoneGap project). 4.Import your project into Android Studio a.Open Android Studio and Choose “Import Project…” b.Navigate to the following directory “platforms/android” within your project path.
